# Steam Wishlist Random Picker

Outil web gratuit pour tirer un jeu au hasard depuis une wishlist Steam.

## Utilisation
1. Rendre sa wishlist Steam publique
2. Entrer son SteamID64
3. Cliquer sur “Pick”

## Limites
- Les profils Steam personnalisés (/id/username) doivent être convertis manuellement
